## Directory Structure

All source code lives under `src/`. Config files stay at root.

```
src/
  app/
    layout.tsx                   # Root layout (existing)
    page.tsx                     # Landing page (existing, keep as-is)
    globals.css                  # Tailwind + semantic tokens (existing)

    demo-cooperativas/           # Static demo (existing, keep as-is)
    demo-financieras/            # Static demo (existing, keep as-is)

    farmer/                      # Farmer PWA (to build)
      layout.tsx                 # Farmer shell + offline detection
      page.tsx                   # Transaction form + sign-in
      history/page.tsx           # Transaction history

    dashboard/                   # Cooperative dashboard (to build)
      layout.tsx                 # Dashboard shell + auth guard
      page.tsx                   # KPIs, charts, traceability
      producers/page.tsx         # Producer list with filters
      producer/[id]/page.tsx     # Individual producer detail
      settings/page.tsx          # Products, invite code, export goals

    scoring/                     # Financiera scoring (to build)
      layout.tsx                 # Scoring shell + auth guard
      page.tsx                   # Portfolio view
      farmer/[id]/page.tsx       # Individual credit profile

    api/
      auth/[...all]/route.ts     # better-auth catch-all
      sync/route.ts              # PWA batch sync endpoint

  lib/
    db.ts                        # Neon connection + Drizzle client
    auth.ts                      # better-auth config + session helpers
    scoring.ts                   # Credit score computation
    sync.ts                      # Batch validation, UUID dedup
    prices.ts                    # Price benchmarks per crop/region
    integrity.ts                 # Anomaly detection, trust scores
    utils.ts                     # cn() helper (existing)

  db/
    schema.ts                    # Drizzle schema (single source of truth)
    index.ts                     # Re-exports Drizzle client
    migrations/                  # Generated by drizzle-kit

  actions/
    transactions.ts              # Server Actions: create, confirm, correct
    farmers.ts                   # Server Actions: register, update profile
    cooperatives.ts              # Server Actions: manage coop, products, goals

  components/
    ui/                          # shadcn/ui (existing)
    landing/                     # Landing page components (existing)

  hooks/
    use-mobile.ts                # Existing

# Root config files
biome.jsonc                      # Lint + format (replaces ESLint + Prettier)
components.json                  # shadcn/ui config
drizzle.config.ts                # Drizzle-kit config (Phase 1)
next.config.mjs                  # Next.js config
tsconfig.json                    # TypeScript config
package.json
```

## Tooling

| Tool | Purpose | Replaces |
|------|---------|----------|
| Biome | Lint + format | ESLint + Prettier |

Biome config: tabs, double quotes, organized imports. Lint excludes `src/components/ui/*` (shadcn-generated).
